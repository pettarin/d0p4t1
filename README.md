# d0p4t1

A Bash script to scrape the [DOP online](http://www.dizionario.rai.it).

* Date: 2016-05-05
* Version: 1.0.0
* License: MIT

## Usage

```
$ ./d0p4t1 from to [sleep]
```

## Example

```
$ ./d0p4t1 64778 64780 
$ ./d0p4t1 64778 64780 5 
```
